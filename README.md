# Ranju Hasan Khan Portfolio - React Version

This is a React.js conversion of the original HTML portfolio website for Ranju Hasan Khan - Full-Stack Developer & International Speaker for World Peace.

## Project Structure

```
src/
├── components/
│   ├── About.js          # About Me section with competencies
│   ├── BackToTop.js      # Back to top button component
│   ├── Contact.js        # Contact and connection section
│   ├── Footer.js         # Footer with links
│   ├── Hero.js           # Hero section with typing animation
│   ├── Navigation.js     # Navigation bar
│   ├── Particles.js      # Animated particles background
│   ├── Peace.js          # World Peace advocacy section
│   ├── Skills.js         # Professional skills section
│   ├── Stats.js          # GitHub statistics section
│   └── TechStack.js      # Technology stack showcase
├── App.js                # Main App component
├── App.css               # App styles
├── index.js              # React entry point
└── index.css             # Global styles

public/
└── index.html            # HTML template

package.json              # Project dependencies and scripts
```

## Features

- **Responsive Design**: Works on all device sizes
- **Smooth Scrolling**: Smooth navigation between sections
- **Animated Hero Section**: Dynamic typing effect in hero text
- **Glitch Animation**: Eye-catching title animation
- **Floating Particles**: Dynamic background particles
- **Grid Background**: Animated grid pattern background
- **Hover Effects**: Interactive card and button hover animations
- **GitHub Integration**: Displays GitHub stats and streak
- **Dark Theme**: Modern dark theme with cyan, magenta, and yellow accents

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```
